Pricing & Plans

Competitive pricing for virtual servers, storage and managed platform services.

Service Components

Platform supports virtual machine hosting, object storage, container orchestration and managed databases. Network traffic charged per GB transferred. Storage includes 99.99% durability and multi-zone replication. All plans include 24/7 monitoring and basic support.

Usage Examples

Deploy 2 vCPU VM with 8GB RAM: $0.04/hour. Store 500GB objects: $11.50/month. Run 1M function invocations: $0.20.

Configuration Snippet

provider "cloudinfra" {
  region = "us-west-1"
  vm_size = "c4.large"
  storage = "s3-standard"
  
  resource "vm_instance" {
    name = "web-server"
    cpu  = 2
    ram  = "8GB"
  }
}
